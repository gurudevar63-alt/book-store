# Introduction
Book Store is a website where users can browse and purchase books online. It has a user-friendly interface that allows users to easily search for books and add them to their shopping cart. 
# Book Store was built using the following technologies:
* HTML
* CSS
* JavaScript
* PHP
* MySQL
# Features
* User registration and login
* Search and filter books by category, author, and title
* Add books to shopping cart and checkout
* Responsive design for mobile and desktop devices
# Installation
To install Book Store, follow these steps:
1.	Clone the repository: git clone https://github.com/rahulemathi/bookstore.git
2. Create a database name as books.
3. Import the database to phpmyadmin 
4. Start the apache and mysql server in xampp
# Usage
To use Book Store, open the index.php file in your web browser. You will be prompted to register or login to access the website.
